# Rev

Roblox account linking, made easy.

## Installation

Install Rev with npm

```bash
npm install @grid-rbx/rev
```

Then initiate it in your index file

```javascript
// CommonJS
const rev = require("@grid-rbx/rev");

// ES6
import rev from "@grid-rbx/rev;
```
